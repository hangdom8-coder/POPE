Data Types

ប្រភេទទិន្នន័យ Data Types

1. Integer (ចំនួនគត់)
$age = 25;
2. Float / Double (ចំនួនទសភាគ)
$price = 12.5;
3. String (អក្សរ)
$name = "Oudom";
4. Boolean (true/false)
$isLogin = true;
5. Array (បញ្ជី)
$colors = array("Red","Green","Blue");
6. Object (Class)
class Car {
  public $color;
}

$myCar = new Car();
$myCar->color = "Red";
7. NULL
$data = null;
8. Resource
$conn = mysqli_connect("localhost","root","","test");
