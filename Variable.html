Variable

Variable (អថេរ) គឺជាទីតាំងមួយសម្រាប់រក្សាទុកទិន្នន័យក្នុងកម្មវិធី។
វាអាចផ្ទុកតម្លៃផ្សេងៗ ដូចជា លេខ អក្សរ ឬ តម្លៃផ្សេងៗ។

1. របៀបប្រកាស Variable

រាល់ Variable ក្នុង PHP ទាំងអស់ត្រូវតែចាប់ផ្តើមដោយសញ្ញាដុល្លារ $ ជានិច្ច។ EX:

<?php
$name = "Heng Oudom"; // ប្រភេទអត្ថបទ (String)
$age = 20;           // ប្រភេទលេខ (Integer)
$gpa = 4.5;          // ប្រភេទលេខទសភាគ (Float)
?>

2. ការដាក់ឈ្មោះ Variable

  1. ត្រូវចាប់ផ្តើមដោយសញ្ញា $
  2. បន្ទាប់ពីសញ្ញា $ ត្រូវតែជា អក្សរ ឬសញ្ញា Underscore (_). មិនអាចចាប់ផ្តើមដោយលេខឡើយ។
  3. ឈ្មោះ Variable ដែលមានអក្សរតូច និងធំ គឺខុសគ្នា។ ឧទាហរណ៍ $color និង $COLOR គឺជា Variable ពីរផ្សេងគ្នា។
